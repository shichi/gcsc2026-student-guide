# GCSC2026 学生ガイド（日本語）

**Global Challenge for Social Innovation Camp 2026**

日本と韓国の大学生が共に社会課題に取り組む4日間のイノベーションキャンプ

[🇰🇷 한국어](README.ko.md) | [🇬🇧 English](README.en.md)

## 🗓 イベント情報

- **日程**: 2026年2月8日（日）〜11日（水・祝）
- **会場**: 国立オリンピック記念青少年総合センター（代々木）

## 🎓 メンタリングについて

### 予約システムの特徴
- **20分単位**のタイムスロット制
- チームは各セッションで**複数のタイムスロット**を予約可能（ただし連続不可）
- Discord上で簡単に予約・キャンセル可能

### メンターセッション（2月9日）
- **Session 1**: 10:00-12:00（6スロット）
- **Session 2**: 13:00-15:00（6スロット）
- **Session 3**: 15:00-17:00（6スロット）
- **Faculty Night**: 20:00-22:00（予約不要・教員巡回）

### 予約方法
チームチャンネルで以下のコマンドを実行：
```
/mentor ui
```
インタラクティブUIでセッション・時間・メンターを選択できます。

詳細は [メンター予約ガイド](mentor-booking/guide.ja.md) を参照。

## 🎯 目標

1. **社会課題の発見と解決**: 日韓共通の社会課題を見つけ、革新的な解決策を提案
2. **国際協働**: 言語・文化を超えたチームワーク
3. **プロトタイピング**: アイデアを形にする実践力
4. **プレゼンテーション**: 最終日に成果を発表
