# GCSC2026 学生ガイド（日本語）

**Global Challenge for Social Innovation Camp 2026**

日本と韓国の大学生が共に社会課題に取り組む4日間のイノベーションキャンプ

[🇰🇷 한국어](README.ko.md) | [🇬🇧 English](README.en.md)

---

## 🗓 イベント情報

- **日程**: 2026年2月8日（日）〜11日（水・祝）
- **会場**: 国立オリンピック記念青少年総合センター（代々木）
- **参加者**: 日本40名 + 韓国40名 = 計80名
- **チーム**: 20チーム（各4名：日本2名＋韓国2名）

## 🚀 クイックスタート

### 1. Discordに参加
招待リンクからGCSC2026 Discordサーバーに参加してください。

### 2. 自己紹介を投稿
`#introductions` チャンネルで自己紹介してください。
```
/set_intro
```
コマンドで自己紹介を登録できます。

### 3. チーム編成
- **日本学生**: 2月8日（日）開会式後にチーム編成
- **韓国学生**: 事前にチーム情報を共有

### 4. メンター予約
チームチャンネルから `/mentor ui` でメンターを予約できます。
詳細は [メンター予約ガイド](mentor-booking/guide.md) を参照。

## 🎯 目標

1. **社会課題の発見と解決**: 日韓共通の社会課題を見つけ、革新的な解決策を提案
2. **国際協働**: 言語・文化を超えたチームワーク
3. **プロトタイピング**: アイデアを形にする実践力
4. **プレゼンテーション**: 最終日に成果を発表
