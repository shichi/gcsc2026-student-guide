# Discord Getting Started Guide

Discord基本操作ガイド

---

## 🎯 What is Discord?

Discordは、チャットとボイスコミュニケーションのためのプラットフォームです。GCSC2026では、チーム内のコミュニケーション、情報共有、メンター予約などに使用します。

---

## 📱 Setup

### 1. アカウント作成

1. [Discord公式サイト](https://discord.com/)にアクセス
2. 「登録」をクリック
3. メールアドレス、ユーザー名、パスワードを入力
4. メール認証を完了

### 2. サーバーに参加

招待リンクからGCSC2026サーバーに参加してください。

### 3. プロフィール設定

1. 左下のユーザー名をクリック
2. 「ユーザー設定」を開く
3. プロフィール画像とユーザー名を設定

**推奨**: ユーザー名は本名または識別しやすい名前にしてください

---

## 🏷️ Server Nickname

サーバー内での表示名を設定できます。

### 推奨フォーマット:
```
[所属大学] 名前 (例: Hitotsubashi Taro / Gachon Minsu)
```

### 設定方法:
1. サーバー名を右クリック
2. 「ニックネームを変更」を選択
3. 新しいニックネームを入力

---

## 📢 Channels Overview

### Public Channels

| チャンネル | 用途 |
|-----------|------|
| `#general` | 全体での雑談・質問 |
| `#announcements` | 重要なお知らせ（運営からの通知） |
| `#introductions` | 自己紹介 |
| `#schedule` | スケジュール・タイムテーブル |
| `#resources` | 資料・リンク共有 |
| `#random` | 自由な話題 |

### Team Channels

チーム編成後、各チーム専用のチャンネルが作成されます：
- `#team-01` 〜 `#team-20`

**これらのチャンネルで:**
- チームディスカッション
- メンター予約 (`/mentor` コマンド)
- ファイル共有
- ボイスチャット

### Voice Channels

ボイスチャンネルをクリックすると、音声通話に参加できます。
- 各チーム専用のボイスチャンネル
- 全体集合用のボイスチャンネル

---

## 🎤 Voice Chat

### 参加方法
1. ボイスチャンネルをクリック
2. マイク・スピーカーの許可
3. 自動的に参加

### 操作
- **マイクミュート**: 左下のマイクアイコン
- **スピーカーミュート**: 左下のヘッドフォンアイコン
- **退出**: 「切断」ボタン

### Tips
- **プッシュトゥトーク**: 設定で有効にすると、特定のキーを押している間だけマイクオン
- **画面共有**: ボイスチャンネル参加中に「画面」アイコンをクリック

---

## 💬 Text Chat Basics

### メッセージ送信
- テキストボックスに入力してEnter
- Shift + Enter で改行

### メンション
- `@ユーザー名` で特定のユーザーに通知
- `@everyone` で全員に通知（使用は控えめに）
- `@here` でオンラインの全員に通知

### リアクション
- メッセージにカーソルを合わせて絵文字アイコン
- クイック投票や同意表明に便利

### ファイル共有
- ドラッグ＆ドロップでファイルアップロード
- 画像は自動的にプレビュー表示

## 🤖 Bot Commands

GCSC2026サーバーには専用Botが導入されています。

### 基本コマンド

```
/set_intro
```
自己紹介を登録

```
/mentor ui
```
メンター予約（インタラクティブUI）

```
/mentor my
```
自分のチームの予約確認

### コマンドの使い方

1. チャットボックスに `/` を入力
2. コマンド候補が表示される
3. 使いたいコマンドを選択
4. 必要なパラメータを入力
5. Enter で実行

詳細は [Commands Guide](commands.md) を参照。

## 🔔 Notifications

### 通知設定

通知が多すぎる場合は、カスタマイズできます：

1. サーバー名を右クリック
2. 「通知設定」を選択
3. 通知レベルを調整

**推奨設定:**
- `#announcements` - すべてのメッセージで通知
- `#team-XX` (自分のチーム) - すべてのメッセージで通知
- その他 - メンションのみ

## ⚠️ Important Rules

1. **尊重**: すべてのメンバーを尊重し、礼儀正しく
2. **言語**: 英語・日本語・韓国語を使用。必要に応じて翻訳ツールを活用
3. **機密情報**: 個人情報や機密情報を公開チャンネルに投稿しない
4. **スパム禁止**: 同じメッセージの繰り返し投稿は避ける
5. **適切な内容**: 不適切な内容の投稿は禁止

## 🆘 Troubleshooting

### 音声が聞こえない
1. スピーカーがミュートになっていないか確認
2. 「ユーザー設定」→「音声・ビデオ」で出力デバイスを確認
3. Discordを再起動

### マイクが動作しない
1. マイクがミュートになっていないか確認
2. 「ユーザー設定」→「音声・ビデオ」で入力デバイスを確認
3. ブラウザ/OSのマイク許可を確認

### チャンネルが見えない
1. サーバーに正しく参加しているか確認
2. チーム編成前の場合、チームチャンネルはまだ表示されません
3. 運営スタッフに問い合わせ
