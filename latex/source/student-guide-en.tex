\section{About Mentoring}

\subsection{Booking System Features}

\begin{itemize}
    \item \textbf{20-minute} timeslot system
    \item Teams can book \textbf{multiple timeslots} per session (but not consecutive)
    \item Easy booking and cancellation via Discord
\end{itemize}

\subsection{Mentor Sessions (February 9)}

\begin{itemize}
    \item \textbf{Session 1}: 10:00-12:00 (6 slots)
    \item \textbf{Session 2}: 13:00-15:00 (6 slots)
    \item \textbf{Session 3}: 15:00-17:00 (6 slots)
    \item \textbf{Faculty Night}: 20:00-22:00 (No booking required - faculty circulates)
\end{itemize}

\subsection{How to Book}

Run this command in your team channel:

\begin{verbatim}
/mentor ui
\end{verbatim}

Interactive UI lets you select session, time, and mentor.

\section{Goals}

\begin{enumerate}
    \item \textbf{Discover and Solve Social Issues}: Find common Japan-Korea social issues and propose innovative solutions
    \item \textbf{International Collaboration}: Teamwork across language and cultural barriers
    \item \textbf{Prototyping}: Practical skills to bring ideas to life
    \item \textbf{Presentation}: Present outcomes on final day
\end{enumerate}
