[🇯🇵 日本語](README.ja.md) | [🇬🇧 English](README.en.md)

## 🗓 행사 정보

- **일정**: 2026년 2월 8일(일)~11일(수·공휴일)
- **장소**: 국립올림픽기념청소년종합센터(요요기, 도쿄)
- **참가자**: 80명 (일본 40명 + 한국 40명)
- **팀**: 20팀 (각 4명: 일본 2명 + 한국 2명)

## 🚀 빠른 시작

### 1. Discord 참여
초대 링크로 GCSC2026 Discord 서버에 참여하세요.

### 2. 자기소개 게시
`#introductions` 채널에서 자기소개를 해주세요.
```
/set_intro
```
이 명령어로 자기소개를 등록할 수 있습니다.

### 3. 팀 편성
- **일본 학생**: 2월 8일(일) 개회식 후 팀 편성
- **한국 학생**: 사전에 팀 정보 공유

### 4. 멘토 예약
팀 채널에서 `/mentor ui`로 멘토 세션을 예약할 수 있습니다.
자세한 내용은 [멘토 예약 가이드](mentor-booking/guide.md)를 참조하세요.
