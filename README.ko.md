# GCSC2026 학생 가이드 (한국어)

일본과 한국 대학생이 함께 사회 과제에 도전하는 4일간의 이노베이션 캠프

[🇯🇵 日本語](README.ja.md) | [🇬🇧 English](README.en.md)

## 🗓 이벤트 정보

- **일정**: 2026년 2월 8일(일) ~ 11일(수・공휴일)
- **회장**: 국립올림픽기념청소년종합센터（요요기）
- **참가자**: 일본 40명 + 한국 40명 = 총 80명（참가 예정 77명）

## 🎓 멘토링에 대하여

### 예약 시스템 특징
- **20분 단위**의 타임슬롯 제도
- 팀은 각 세션에서 **여러 타임슬롯** 예약 가능（단, 연속 불가）
- Discord에서 간편하게 예약・취소 가능

### 멘토 세션（2월 9일）
- **Session 1**: 10:00-12:00（6슬롯）
- **Session 2**: 13:00-15:00（6슬롯）
- **Session 3**: 15:00-17:00（6슬롯）
- **Faculty Night**: 20:00-22:00（예약 불필요・교원 순회）

### 예약 방법
팀 채널에서 다음 명령어를 실행:
```
/mentor ui
```
인터랙티브 UI에서 세션・시간・멘토를 선택할 수 있습니다.

자세한 내용은 [멘토 예약 가이드](mentor-booking/guide.ko.md) 참조.
