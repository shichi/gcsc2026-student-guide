## ğŸ“ About Mentoring

### Booking System Features
- **20-minute** timeslot system
- Teams can book **multiple timeslots** per session (but not consecutive)
- Easy booking and cancellation via Discord

### Mentor Sessions (February 9)
- **Session 1**: 10:00-12:00 (6 slots)
- **Session 2**: 13:00-15:00 (6 slots)
- **Session 3**: 15:00-17:00 (6 slots)

Run this command in your team channel:
```
/mentor ui
```
Interactive UI lets you select session, time, and mentor.
