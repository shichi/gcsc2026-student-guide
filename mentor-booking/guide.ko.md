## 📋 시스템 개요

GCSC2026의 멘토 예약은 **20분 단위 타임슬롯 제도**를 채택하고 있습니다.

### 타임슬롯의 특징
- 각 2시간 세션을 **6개의 20분 슬롯**으로 분할
- 팀은 각 세션에서 **여러 타임슬롯** 예약 가능 (단, 연속 불가)
- 각 멘토는 하나의 타임슬롯에서 **1팀만** 대응
- Discord에서 간편하게 예약・취소 가능

## 💬 Discord 명령어

### 1. 인터랙티브 UI 예약 (권장)

팀 채널(`#team-01`~`#team-20`)에서:

```
/mentor ui
```

실행하면 단계별로 예약할 수 있는 UI가 표시됩니다:

1. **세션 선택** → 드롭다운에서 선택
2. **타임슬롯 선택** → 이용 가능한 시간대 선택
3. **멘토 선택** → 예약 가능한 멘토 선택
4. **확인** → 예약 내용을 확인하고 확정

### 2. 직접 예약 (고급 사용자용)

```
/mentor book session:session_1 timeslot:10:00-10:20 mentor:Masatoshi KANEKO
```

**파라미터:**
- `session` - 세션 이름 (`session_1`, `session_2`, `session_3`)
- `timeslot` - 타임슬롯 (예: `10:00-10:20`)
- `mentor` - 멘토 이름 (알파벳 표기)

### 3. 예약 취소

```
/mentor cancel session:session_1 timeslot:10:00-10:20
```

### 4. 우리 팀의 예약 확인

```
/mentor my
```

우리 팀이 예약한 타임슬롯과 멘토가 목록으로 표시됩니다.

### 5. 세션 전체의 예약 상황 확인

```
/mentor status session:session_1
```

지정한 세션의 전체 타임슬롯 예약 상황을 확인할 수 있습니다.

## ⚠️ 중요한 규칙

1. **같은 세션 내에서 여러 예약 가능**: 각 팀은 하나의 세션 내에서 여러 타임슬롯을 예약할 수 있습니다
   - ✅ 예: Session 1의 10:00-10:20과 10:40-11:00을 예약 (OK)
   - ❌ 예: Session 1의 10:00-10:20과 10:20-10:40을 예약 (NG - 연속)

2. **연속되는 타임슬롯은 NG**: 같은 세션 내에서 연속되는 타임슬롯 (예: 10:00-10:20과 10:20-10:40)의 예약은 불가능합니다
   - 적어도 하나의 타임슬롯을 비워야 합니다

3. **같은 타임슬롯에서 중복 예약은 NG**: 같은 타임슬롯에서 같은 멘토를 여러 팀이 예약할 수 없습니다

4. **선착순**: 예약은 선착순. 인기 있는 멘토・시간대는 일찍 예약을

5. **알파벳 입력**: 멘토 이름은 알파벳 표기로 입력 (예: `Masatoshi KANEKO`)
   - 인터랙티브 UI (`/mentor ui`)를 사용하면 선택만 하면 됩니다

6. **Faculty Night는 예약 불가**: 20:00-22:00의 Faculty Night는 교원이 각 팀을 순회합니다

## ❓ FAQ

### Q1: 같은 세션에서 여러 멘토를 예약할 수 있나요?

**A:** 네, 가능합니다. 하나의 세션 내에서 여러 타임슬롯을 예약할 수 있습니다. 단, **연속되는 타임슬롯의 예약은 불가능합니다**. 적어도 하나의 타임슬롯을 비워야 합니다.

**예:**
- ✅ OK: Session 1의 10:00-10:20 (Mentor A)와 10:40-11:00 (Mentor B)
- ❌ NG: Session 1의 10:00-10:20 (Mentor A)와 10:20-10:40 (Mentor B) ← 연속

### Q2: 다른 세션에서 같은 멘토를 예약할 수 있나요?

**A:** 네, 가능합니다. Session 1, Session 2, Session 3은 각각 독립적이므로, 다른 세션이라면 같은 멘토를 예약할 수 있습니다.

### Q3: 예약 후에 시간을 변경하고 싶은 경우?

**A:** 먼저 `/mentor cancel`로 예약을 취소한 다음, 새로운 타임슬롯으로 `/mentor book` 또는 `/mentor ui`를 실행하세요.

### Q4: 멘토 이름 입력이 어렵습니다

**A:** `/mentor ui` 명령어를 사용하면 드롭다운에서 선택만 하면 예약할 수 있습니다. 직접 예약의 경우 복사・붙여넣기도 이용할 수 있습니다.

### Q5: Faculty Night는 어떻게 예약하나요?

**A:** Faculty Night (20:00-22:00)는 예약제가 아닙니다. 교원이 각 팀을 순회하여 지원을 제공합니다.

## 💡 효과적인 멘토링 팁

- **사전 준비**: 멘토링 전에 팀 내에서 질문 사항을 정리해두기
- **20분을 효과적으로 활용**: 시간이 한정되어 있으므로 요점을 정리하여 상담하기
- **여러 세션 활용**: 하나의 세션만이 아니라 여러 세션에서 멘토를 예약하는 것도 고려
- **다양한 관점**: 다른 배경의 멘토에게 상담하여 다양한 관점 얻기
